🌐 Public Discovery:
• Agent will be included in .well-known/agent.json
• Anyone on the internet can discover this agent
• Good for: customer support, public APIs, developer tools
🏢 Internal Discovery:
• Agent discoverable only within your organization
• Not included in public .well-known file
• Good for: HR, finance, internal tools, employee services
🔒 Private:
• No automatic discovery - completely private
• Must be configured manually by teams that need it
• Good for: sensitive systems, admin tools, testing agents
